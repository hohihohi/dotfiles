# Key binds

|command|description|origin|
|:---:|:---|:---:|
|ctrl+p|1つ前のコマンド履歴を見る|Linux|
|ctrl+n|1つ先のコマンド履歴を見る|Linux|
|ctrl+r|コマンド履歴の検索|Linux|
|ctrl+f|1個右へカーソルを移動|Linux|
|ctrl+b|1個左へカーソルを移動|Linux|
|opt+f|単語単位で右へカーソルを移動|Linux|
|opt+b|単語単位で右へカーソルを移動|Linux|
|ctrl+a|行頭へカーソルを移動|Linux|
|ctrl+e|行末へカーソルを移動|Linux|
|ctrl+t|1つ前と現在の文字を入れ替える|Linux|
|ctrl+h|カーソルの左側の文字を削除|Linux|
|ctrl+d|カーソルの文字を削除|Linux|
|ctrl+w|カーソルより左側を単語単位で削除|Linux|
|opt+d|カーソルより右側を単語単位で削除|Linux|
|ctrl+u|カーソルより左側全部を削除|Linux|
|ctrl+k|カーソルより右側全部を削除|Linux|
|ctrl+y|貼付け 上記ctrl-w,ctrl-kなどで削除した文字列をペーストできる|Linux|
|ctrl+l|画面のクリア|Linux|
|ctrl+c|実行中のプログラムを強制終了|Linux|
|ctrl+d|終了|Linux|
|ctrl+s|画面の更新を止める|Linux|
|ctrl+q|画面の更新を再開|Linux|
